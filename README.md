# exteraGram_Plugins

Этот репозиторий содержит плагины для exteraGram на android. для установки скачайте нужный вам плагин, отправте его в любой чат, а затем откройте его и установите

This repository contains plugins for exteraGram on Android. To install, download the plugin you need, send it to any chat, then open it and install it.
